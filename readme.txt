Easily embed a list of web apps in a wordpress blog with value added information linked to each web app.  Visit http://www.cloudsurfing.com/widgets

== Description ==

Embed a list of web apps in your blog post or in your blog right hand menu.  The Slideshow widgets and Logo Cloud widget work well for the posts and the Column List widget and Logo Cloud widget work well for columns or footer areas.   

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `cloudsurfing-widget’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go in Appearance > widgets section, plugin will be listed here
4. Click on Add link adjecent to Cloudsurfing plugin
5. Provide url of list that you get from http://www.cloudsurfing.com/widgets
6. Select the location on your wordpress blog where you want it shown
7. Click on "Save Widget"

== Frequently Asked Questions ==

= From where i can get the url for plugin =

You can get url from http://www.cloudsurfing.com/widgets

1. Visit http://www.cloudsurfing.com/widgets/ and select option "your wordpress blog"
2. Add sites which you want to display
3. Select one of the widget types (e.g. slideshow, list of logos) and click on "Get URL for widget plugin" button
4. Copy the url displayed in textarea just below the "Get URL for widget plugin" button
